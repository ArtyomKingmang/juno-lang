## Features of Juno

- high speed
- Cross-compile

## How to build Juno?
so, for get started we need to install deps: <br><br>
**For archlinux // arch-based:**
```shell
[root@pc: ~] # pacman -S ruby
[root@pc: ~] # pacman -S llvm
```

**For debian // debian-based:**
```shell
[root@pc: ~] # apt install ruby
[root@pc: ~] # apt install llvm
```
